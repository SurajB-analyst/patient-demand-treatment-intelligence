# Patient Demand & Treatment Intelligence Dashboard

## Overview

Built an executive healthcare dashboard to analyze patient demand drivers, treatment utilization, repeat-user behavior, and resource planning using multi-table operational data.

## Business Objective

Help hospital management optimize staffing, outpatient redirection, treatment readiness, and patient flow decisions.

## Tools Used

* Microsoft Excel 2024
* Power Query
* Power Pivot
* Pivot Tables
* DAX Measures
* Slicers
* Conditional Formatting

## Key Questions Answered

* What are the top patient complaint reasons?
* Which age groups generate highest demand?
* What medication types are most used?
* Which complaints create repeat visits?
* Which complaints are high-risk?
* Which cases can shift to outpatient care?

## Key Insights

* Abdominal pain was the highest-volume complaint.
* Patients aged 51+ created the largest admission/surgery burden.
* Repeat demand concentrated in recurring complaint groups.
* Stable high-volume complaints may be redirected to outpatient channels.
* High-risk complaints need faster triage readiness.

## Business Recommendations

* Create fast-track pathways for high-volume low-risk complaints.
* Improve chronic care follow-up for repeat users.
* Increase staffing for senior demand groups.
* Forecast medication inventory from usage patterns.
